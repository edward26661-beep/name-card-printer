# 自动化席卡打印系统（Excel版）

一个基于 Python + Word 自动化的席卡批量打印工具。

## 功能
- 从 Excel 读取姓名列表
- 根据姓名字数自动匹配 Word 模板
- 支持试打确认，再批量打印
- 适用于会议 / 培训 / 活动席卡

## 运行环境
- Windows
- 已安装 Microsoft Word
- Python 3.8+

## 安装依赖
```bash
pip install -r requirements.txt
